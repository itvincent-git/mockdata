# ## Install

```shell
npm install json-server
```



## Try

Pass it to JSON Server CLI

```shell
$ npx json-server db.json
```

Get a REST API

```shell
$ curl http://localhost:3000/todos/1
{
  "id": "1",
  "task_name": "penatibus et magnis",
   ...
}
```
